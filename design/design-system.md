# Design System（Office Hase）

## Colors

- Gold (Accent): #bfa14a
- Text: #222 / #333 / #444 / #555 / #666 / #777
- Background: #fff / #fafafa / #f6f8fa / #f5f5f5
- Borders: #e0e0e0 / #e5e5e5 / #eee / #ddd
- FV Gradient: #f7f8f9 → #f6f8fa → #f3f4f6

## Typography

### Font Family

- Noto Sans JP, Hiragino Kaku Gothic ProN, Meiryo, sans-serif

### Font Sizes

- 10: logo sub
- 14: notes (strength/handling/cta)
- 15: fv note / card body
- 16: base body / section lead
- 18: header title / handling h3 / cta lead (mobile)
- 20: service h3
- 24: fv h2
- 28: section h2
- FV H1: clamp(32, 4vw, 40) / mobile 26

### Line Heights

- Base: 1.7
- Long text: 1.9
- Note: 2.0
- Titles: 1.4
- Lists: 1.7〜1.8

## Spacing Scale

- 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 56 / 64 / 80 / 96
- Section padding: 96 (tablet 80, mobile 64)
- Card padding: 48x40 (tablet 40x32, mobile 32x24)
- Grid gap: 40 (tablet 32, mobile 24)

## Radius

- 8px (FV / cards / button)
